Para que o código seja executado corretamente, será necessária importação das bibliotecas
'Glymur' e 'scikit-image' para o ambiente que você estiver utilizando. É necessário que
isto seja feito via Anaconda (Não importe estas bibliotecas via pip!).

Glymur: a Python interface for JPEG 2000: https://glymur.readthedocs.io/en/latest/introduction.html

scikit-image: image processing in Python: https://scikit-image.org/docs/stable/install.html

Obs.: Não é necessária a importação das bibliotecas Numpy e Pillow, pois elas já se encontram
inseridas no pacote 'scikit-image'.

Os arquivos lossless.py e lossy.py encontram-se prontos para execução, uma vez instaladas as
bibliotecas. Como este é um exemplo simples do que virá a ser a aplicação, ela ainda não apresenta
elementos de interação com o usuário, portanto, basta executar cada um dos arquivos que os resultados
gerados pelas compressões serão exibidos na tela.
